About Rozi Roti

Rozi Roti is the Jobs Planet for the CosmoX Digital Infrastructure Operating System (DIOS). It operates as an independent portal — runnable standalone or integrated into CosmoX shared services.

Architecture

  • TypeScript + Fastify + Zod + Vitest
  • Hexagonal architecture (ports/adapters)
  • 89 API endpoints, 40 database tables
  • 89 integration tests passing
  • Dual-mode: standalone or CosmoX-integrated

Governance

  • ARCHITECTURE_RULES.md — service boundaries, identity, region neutrality
  • IDENTITY_RULES.md — coreSubjectId vs subjectId
  • AI_CONSTITUTION.md — guardrails for all 12 AI agents
  • PRODUCTION_GAPS.md — honest documentation of what's not production-ready

  8. Monetization (Subscription Tiers, Billing Hooks)
  9. AI Agents (12 agents across 4 categories)
